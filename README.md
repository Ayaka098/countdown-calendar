# 嵐カウントダウンカレンダー

嵐の活動終了日（2026-05-31）までの残り日数を、日めくりカレンダー風に表示するシングルページアプリです。Next.js + TypeScript (App Router) で構築し、Vercelデプロイ想定の静的サイトとしてビルドされます。

## 特徴
- クライアントサイド計算で「あと○○日」を表示（時間は非表示、リロード時に更新）。
- 右下からめくれる紙のフリップアニメーション（初回ロード時のみ、約5.1s）。
- メンバーカラー5色のポップな背景。☆が均等に各色含まれるようランダム配置で毎回変化。
- レイアウトはスマホ優先・中央の大きなカレンダーカードのみ。
- 依存はNext.js標準＋CSS Modulesのみ。

## 開発
```bash
npm install
npm run dev
# http://localhost:3000
```

## ビルド
```bash
npm run build
npm run start  # 本番確認
```

## デプロイ（Vercel）
- このリポジトリをVercelに接続し、フレームワークに「Next.js」を選択するだけでOKです。
- 環境変数や追加の設定は不要です（静的プリレンダー `/` のみ）。

## 実装メモ
- 残日数は`2026-05-31`固定。`app/page.tsx`の`END_DATE`で管理。
- 背景の☆はロード時にランダム生成し、5色が同数含まれるようにしています。
- アニメーションや装飾は`app/page.module.css`で定義。Tailwind等は未使用（グローバルは初期設定のまま）。***
